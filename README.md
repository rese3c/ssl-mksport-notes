# ssl-mksport-notes

## 项目简介

本仓库用于归档和发布多个独立的 HTML 页面。这些页面内容涉及 SSL 相关技术笔记、工具使用记录以及部分测试环境的快照。仓库本身不针对任何具体域名或网站，仅作为个人技术资料的静态展示与备份。

## 目录说明

```
├── pages/          # 存放所有 HTML 页面源文件
├── assets/         # 页面引用的静态资源（图片、样式、脚本等）
├── README.md       # 本文件
└── .gitignore      # Git 忽略规则
```

- `pages/`：每个 HTML 文件代表一个独立的归档条目，文件名通常反映主题或日期。
- `assets/`：按需组织子目录，与页面文件保持相对路径引用关系。

## 页面归档说明

- 所有页面均为纯前端 HTML，不依赖后端服务或数据库。
- 页面内容可能包含个人笔记、调试记录、实验性代码片段等，仅供技术参考。
- 部分页面可能使用外部 CDN 加载库（如 highlight.js、MathJax 等），请确保网络可达。
- 页面之间的链接均为相对路径，可在本地直接打开查看。

## 维护说明

- 本仓库不定期更新，新增页面或修正已有内容。
- 欢迎通过 Issue 提交建议或报告排版问题，但请注意本仓库不针对特定网站或服务。
- 若需引用或转载页面内容，请注明出处并保持原始链接有效。
